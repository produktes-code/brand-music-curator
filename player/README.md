## 🎯 Overview / Resumen

### English
**Brand Music Curator Player** is the client application for the B2B sensory neuro-architecture audio platform. It operates as a local desktop player that automates store playlists, handles local cache storage for network resiliency (zero-silence), provides local supervisor locks (PIN `1234`), and integrates a copyright inspector protection system (SGAE Shield).

### Español
**Brand Music Curator Player** es la aplicación cliente para la plataforma de audio de neuro-arquitectura sensorial B2B. Funciona como un reproductor local que automatiza listas, dispone de caché offline para cortes de red (cero-silencio), bloqueo de supervisor (PIN `1234`) y un escudo protector ante inspectores de derechos de autor (SGAE Shield).

## 🏗️ Technical Architecture / Arquitectura Técnica

*   **Frontend Interface:** Built using **React 19** with custom Stitch design tokens to deliver a high-performance *Glassmorphism* dashboard.
*   **Desktop Shell:** Structured in **Electron** to bundle as a native standalone client for macOS and Windows.
*   **Database Engine:** Employs **Better-SQLite3** locally to persist player states, lists, and schedules across system power blackouts.

## ⚙️ Core Modules & Features

### 🔒 B2B Player Lock
*   A padlock icon on the top-right header locks the player interface, deselecting and disabling local track skips, playlist switches, and volume controls for store staff.
*   Unlock using the supervisor-level PIN: **`1234`**.

### 🎧 Mixes Manager (Smart DJ)
*   Formulate dynamic music recipes by choosing base and contrapuntal genres from **10 real B2B styles** (*Indie Pop, Deep House, Chillout, Jazz, Rock, Pop Comercial, Latino/Urbano, Clásica, Blues, Soul*).
*   Adjust blending ratios, energy thresholds, ad insertion schedules, and explicit content blocks.

### 📅 Dayparting Matrix
*   Weekly automation grid (7 days × 4 blocks) to schedule mixes. Audio transitions are smoothed automatically using a native 7-second crossfade.

### 🛡️ SGAE Shield (Legal Compliance)
*   A red panic button that instantly purges system memory, silences commercial catalog tracks, and shifts to local royalty-free music files (exemption code **`RF-101`**), outputting immutable local audit logs.
